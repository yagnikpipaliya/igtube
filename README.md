# IGTUBE
Igtube is work as downloader for instagram and youtube. In youtube you can download upto 1080p quality.

## IGTUBE Download Features
- Youtube :
  - Youtube Video upto 1080p quality
  - Youtube Shorts
- Instagram :
  - Post
  - Stories
  - Single Post 
  - All Post
